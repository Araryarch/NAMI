# Nami VS Code Extension

This directory will contain the VS Code extension for Nami language support.

The extension will be implemented in task 11 (Create VS Code Extension) and will include:

- TextMate grammar integration for syntax highlighting
- LSP client connection and management
- File association for .nm files
- Code snippets for common Nami patterns
- Build task integration with Nami CLI
- Extension settings for LSP configuration

## Structure

- `package.json` - Extension manifest
- `src/` - TypeScript source code
- `syntaxes/` - TextMate grammar files
- `snippets/` - Code snippets
- `tasks/` - Build task definitions
