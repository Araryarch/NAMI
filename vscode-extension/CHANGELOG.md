# Change Log

All notable changes to the "nami-language" extension will be documented in this file.

## [0.1.0] - 2024-01-XX

### Added
- Initial release
- Syntax highlighting for NAMI language
- Support for `.nm` file extension
- Code snippets for common patterns:
  - Function declarations
  - Control flow statements
  - Array operations
  - Object literals
  - Async/await
- Auto-closing pairs for brackets and quotes
- Comment toggling support
- Bracket matching
- Code folding
- NAMI Dark theme
- Language configuration:
  - Indentation rules
  - Word patterns
  - Folding markers

### Syntax Support
- Keywords (if, else, for, while, function, class, etc.)
- Operators (arithmetic, comparison, logical, bitwise)
- Strings (single, double, template literals)
- Numbers (decimal, hex, binary, octal)
- Comments (line and block)
- Built-in functions (println, print, input, Math.*, etc.)
- Array methods (map, filter, reduce, etc.)
- String methods (split, join, substring, etc.)
- Constants (true, false, null, undefined, NaN, Infinity)
